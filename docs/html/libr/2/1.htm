DOS/32 Advanced DOS Extender - D32A C/C++ Run-Time Library Reference


2.01 - DOS/32 Advanced D32A Library function - d32a_detect_extender()

Synopsys:

#include <d32a.h>
int d32a_detect_extender(void);


Description:

Detects DOS/32 Advanced DOS Extender.


Returns:

TRUE if the application is running under DOS/32 Advanced DOS Extender, FALSE otherwise (if the DOS Extender under which the application runs is not DOS/32 Advanced).
