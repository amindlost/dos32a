5.0 - Extended Mouse functions - Overview

Mouse functions that have been extended and listed below are available through software interrupt 33h (INT 33h) in protected mode. Most of other mouse functions (such as 0000h, 0001h, 0002h and so on) will also work when called by your program from protected mode. The function number is placed in the AX register, and no error codes are reported on function's return. Note that if the DOS Extender could not detect a mouse driver at startup, these functions will not be available in protected mode (INT 33h interrupt handler will simply return with an IRETD instruction).
