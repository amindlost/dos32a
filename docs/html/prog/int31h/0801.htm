2.44 - DPMI function 0801h - Free Physical Address Mapping

Releases a mapping of physical to linear addresses that was previously obtained with the Physical Address Mapping DPMI function 0800h.

 

In: AX = 0801h
BX:CX = linear address returned by physical address mapping call
Out:

if successful:
CF clear

if failed:
CF set
AX = error code

8025h - invalid linear address

Notes:

a) The client should call this function when it is finished using a device previously mapped to linear addresses with the Physical Address Mapping DPMI function 0800h.
