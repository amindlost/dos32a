2.60 - API function 07h - Get Critical Handler Entry Point

 

In: AL = 07h
Out: CX:DX = 16:16 selector:offset pointer to critical handler routine

Notes:

a) Selector returned by this function should NOT be modified or freed by the user.

b) The selector and offset are returned as a 16-bit values.
