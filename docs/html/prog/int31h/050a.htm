2.35 - DPMI function 050Ah - Get Memory Block Size and Base

Returns the size of a memory block that was previously allocated with DPMI function 0501h.

 

In: AX = 050Ah
SI:DI = memory block handle
Out:

if successful:
CF clear
SI:DI = size of memory block (bytes)
BX:CX = base address of memory block

if failed:
CF set
AX = error code

8023h - invalid handle
