4.07 - VBE function 4F0Ah - Get Protected Mode Interface

 

In: AX = 4F0Ah
BL = 00h - return protected mode table
Out: ES:EDI = pointer to protected mode table data
ECX = length of table
AL = 4Fh (if function supported by VESA BIOS)
AH = 00h (if function was successful)
