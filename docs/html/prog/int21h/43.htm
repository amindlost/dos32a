3.20 - DOS function 43h - Change File Attributes

 

In: AH = 43h
AL = function code
ECX = desired file attributes
DS:EDX = pointer to ASCIIZ file name
Out:

if successful:
CF clear
ECX = current attributes

if failed:
CF set
EAX = DOS error code
