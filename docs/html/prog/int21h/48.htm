3.22 - DOS function 48h - Allocate DOS Memory

 

In: AH = 48h
EBX = paragraphs to allocate
Out:

if successful:
CF clear
EAX = selector of allocated memory block

if failed:
CF set
EAX = DOS error code
EBX = maximum paragraphs available
